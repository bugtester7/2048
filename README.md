# 2048
_Simple_ script to have a _relatively_ high score in 2048!
_Click_ [here][gm] for the game!

### Requirements:  
You need to install _selenium_ before running the script:  
```python  
sudo pip install selenium  
```  
To run the script:  
```python  
python 2048.py  
```
